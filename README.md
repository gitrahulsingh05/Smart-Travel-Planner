🌍 Smart Travel Planner

A responsive travel website built using **HTML5** and **CSS3**. The project helps users explore destinations, discover travel packages, and plan their dream vacations with a modern and user-friendly interface.

 🚀 Features

* 🏠 Attractive Home Page
* 🌎 Destinations Page
* ✈️ Trips Page
* 📞 Contact Section
* 🖼️ Travel Gallery
* ⭐ Top Experiences Section
* 🎯 Call-to-Action Sections
* 🧭 Responsive Navigation Bar
* 📱 Clean and Responsive Design

 🛠️ Technologies Used

* HTML5
* CSS3
